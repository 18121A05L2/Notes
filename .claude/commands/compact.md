# /compact

Summarize the current conversation and persistent context to reduce token usage while retaining key decisions and progress.

## Usage
`$ /compact`

## Instructions
1. Review the conversation history.
2. Identify architectural decisions, finalized code snippets, and active todos.
3. Replace the current session context with a concise summary.
4. Update `CLAUDE.md` and `.claudecode.md` if any permanent changes were decided.
