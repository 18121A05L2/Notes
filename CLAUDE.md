# Project Overview
This repository is a personal workspace and collection of notes focused on Blockchain, Ethereum, Rust, AI, and Web Development. It serves as a study guide and reference for technical concepts and projects.

## Directory Structure
- `Ethereum/`: Solidity, Foundry, EVM, and DeFI notes.
- `Rust/`: Rust programming basics, Solana, and other Rust experiments.
- `AI/`: Information on AI tools, Claude Code, and agentic workflows.
- `TechStack/`: General technical skills (DSA, TypeScript, GitHub).
- `fullstack/`: Web development notes (Frontend, Backend, DevOps).
- `miscellaneous/`: General todos and quick snippets.

## Guiding Principles
- **Conciseness**: Notes should be brief and point-based.
- **Categorization**: Keep related information in specific directories.
- **File naming**: Use descriptive names; prefers `.txt` or `.md`.
- **Learning Goals**: Current focus includes NestJS, Uniswap, and Solana development.

## Common Operations
- **Note Management**: Add new files to the relevant subdirectory. Link to existing notes if they are related.
- **Style**: Use markdown for structure within `.md` files. For `.txt`, follow existing bulleted conventions.

## AI Configuration (.claude)
This project follows Claude Code standards for persistent context and tool configuration:
- `CLAUDE.md`: Always-on project standards and overview.
- `.claudecode.md`: Personal agent instructions and memory.
- `.claude/skills/`: Task-specific expertise (e.g., `blockchain-audit`).
- `.claude/commands/`: Custom slash commands (e.g., `/compact`).
- `.claude/settings.json`: Project-level AI tool permissions and hooks.
- `.claude/managed-settings.json`: Enterprise-level mandatory standards and compliance (Highest priority).
