-> git reset --soft HEAD~1
    - The --soft option means that you will not lose the uncommitted changes you may have.
    - HEAD = current commit
    - HEAD~1 = one commit before
    - HEAD~2 = two commits before, etc.
    - git revert <commit to revert>
